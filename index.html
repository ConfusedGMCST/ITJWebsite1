Disciplines

Computer Engineering - a general field, including both Software & Hardware development

Cyber Security - is what keeps all of the other fields files secure from all malware

Information Systems - is the field in which information is stored, processed, collected, and distributed

Informational Technologies - is a set of fields that includes computer systems, software, programming, languages, and data storage

Software Engineering - is the field for all things programming. It has to do with Desktop, Mobile, Console, Smart Fridge, any applications

Data Sciences - is a general term for the study of all facets of Data Processing

Software Engineering

Required Education/Pathway: in order to get a job in Software Engineering, a good portfolio of projects and a bachelor's degree should land you a job easy and quickly.

Local colleges that include CCM, TCNJ,Rutgers, and others offer comprehensive Computer Science classes that help with your career in Software Engineering.

Typical responsibilities for any Software Engineer may include: Frontend programming, Backend programming, Teamwork, Creativity, Intense deadlines

Companies that Software Engineers work at can range from large corporations to small non-profit teams. The salary is usually rewarding, around ~$45,000 to $520,000, but fluxuates based off of responsibilites and size of the company/your position & location.
